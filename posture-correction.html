Real-Time Posture Correction & Kinematic Alignment

A real-time yoga pose evaluation and correction system built in Python using Google MediaPipe for skeletal landmark extraction and a custom Jacobi Singular Value Decomposition (SVD) solver for spatial alignment.

Python Google MediaPipe Jacobi SVD Solver OpenCV Kinematic Mapping

Key System Architecture

MediaPipe Pose Estimation

Extracts 33 3D body landmark coordinates at 30+ FPS directly from standard webcam feeds without specialized hardware.

Custom Jacobi SVD Solver

Implemented a linear algebra Jacobi SVD solver to compute optimal rigid transformations (rotation & translation) for mapping user skeletons onto ground-truth poses.

Joint-Specific Visual Feedback

Calculates angular error bounds per joint, providing dynamic visual overlays and real-time posture correction cues on screen.
