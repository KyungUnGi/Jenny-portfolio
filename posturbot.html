PosturBot — Wearable Cervical Posture Monitor

A dual-IMU wearable system designed from scratch to detect cervical forward head posture (FHP) in real time using relative angle measurement, eliminating false alerts from full-body lean.

System Architecture

Dual-IMU Cancellation

Measures relative tilt between ear (0x68) and neck C7 (0x69). A complementary filter (α=0.90 gyro, α=0.10 accel) cancels out full-body lean.

4-State Haptic Escalation

When angle > 15°, a 2N2222-driven coin motor escalates alerts based on duration: WARN (0s) → GENTLE (10s) → STRONG (30s) → CRITICAL (60s).

BLE Dashboard & Calibration

Streams real-time cervical angle over Web Bluetooth to a JS/Chart.js web dashboard with session statistics, CSV export, and wireless baseline reset.
